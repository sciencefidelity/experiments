# Vite React Template

A full featured template for developing React apps with Vite. Includes config for Eslint, Prettier and Stylelint, and Jest with Testing Library.

### Commands

```shell
yarn dev
```

```shell
yarn build
```

```shell
yarn serve
```

```shell
yarn clean
```

```shell
yarn lint
```

```shell
yarn fix
```

```shell
yarn test
```

```shell
yarn coverage
```
